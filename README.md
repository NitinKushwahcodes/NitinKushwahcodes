### Projects

&nbsp;

**[TaskFlow](https://taskflowxt.netlify.app/) — Full Stack Task Management System**
`Node.js` `Express` `MongoDB` `Vanilla JS` `JWT`

A production-deployed task management system built with a full MVC architecture. Supports three user roles with JWT authentication and RBAC, 15+ REST API endpoints with filtering, pagination, and priority sorting, and a security layer using Helmet, bcrypt, CORS, and rate limiting. Built and deployed independently — live at [taskflowxt.netlify.app](https://taskflowxt.netlify.app/)

&nbsp;

**[AI Interview Assistant](https://github.com/NitinKushwahcodes) — MERN + LLM Integration** *(In Development)*
`React` `Node.js` `Express` `MongoDB` `GPT/Gemini API` `Puppeteer` `Zod`

End-to-end mock interview platform with real-time AI feedback, automated PDF report generation, file upload handling via Multer, and strict schema validation across all routes. Frontend built with React using Context API and custom hooks for state management.

&nbsp;

**[IndiaQuant MCP](https://github.com/NitinKushwahcodes) — Stock Intelligence Backend**
`Node.js` `Express` `SQLite` `NLP` `Financial APIs`

A backend system for real-time stock market intelligence — integrates NSE/BSE data feeds, runs RSI-based trading signals combined with NLP sentiment analysis on financial news, and computes full Black-Scholes Greeks for options analytics. Processes 500+ datapoints per refresh cycle in under 300ms.
